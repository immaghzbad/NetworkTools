## **Network Tools** 🌐

A powerful all-in-one desktop utility for **network diagnostics** and **security testing**, built with C# (WPF).

### ✨ Features

* ⚡️ Ping hosts (ICMP)
* 🌐 DNS Lookup
* 🔐 Port Scanner (Open / Closed / Filtered ports)
* 🕵️ Subdomain Finder
* 🌍 Traceroute
